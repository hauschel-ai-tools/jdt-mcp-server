Eclipse JDT MCP Server 44 Tools

Ein Eclipse-Plugin, das einen MCP (Model Context Protocol) Server einbettet, um Eclipse JDT-Funktionen für KI-Coding-Assistenten wie Claude Code und Cursor bereitzustellen.

P2 Update Site

https://naturzukunft.codeberg.page/jdt-mcp-server/

Installation in Eclipse

  1. Eclipse starten
    Eclipse IDE for Java Developers (2025-12 oder neuer)
  2. Install New Software
    Menü: Help > Install New Software...
  3. Update Site hinzufügen
    Auf Add... klicken und die URL eingeben
  4. Plugin installieren
    "Eclipse JDT MCP Server" anhaken und Next
  5. Eclipse neu starten
    Der MCP Server startet automatisch auf Port 51000+

KI-Assistent konfigurieren

Claude Code (empfohlen: HTTP Transport)

In ~/.claude/settings.json:

{
  "mcpServers": {
    "eclipse-jdt": {
      "url": "http://localhost:51000/mcp",
      "type": "http"
    }
  }
}

Alternative: SSE Transport

{
  "mcpServers": {
    "eclipse-jdt": {
      "url": "http://localhost:51000/sse",
      "transport": "sse"
    }
  }
}

Hinweis: Port in Eclipse-Konsole prüfen: [JDT MCP] HTTP Server started on port XXXXX

Intelligente Maven-Integration

🚀 Automatische Erkennung

  • Maven Wrapper: ./mvnw wird automatisch bevorzugt, falls vorhanden
  • JAVA_HOME: Wird basierend auf Projekt-Java-Version gesetzt (SDKMAN, /usr/lib/jvm)
  • Multi-Module: jdt_list_projects zeigt Maven-Metadaten (groupId, artifactId, parent, modules)
  • Strukturierter Output: Test-Ergebnisse als JSON mit testsRun, failures, failedTests[]
