Eclipse JDT MCP Server

Ein Eclipse-Plugin, das einen MCP (Model Context Protocol) Server einbettet, um Eclipse JDT-Funktionen für KI-Coding-Assistenten bereitzustellen.

P2 Update Site

https://naturzukunft.codeberg.page/jdt-mcp-server/

Installation in Eclipse

  1. Eclipse öffnen
    Eclipse IDE for Java Developers (2024-03 oder neuer) starten
  2. Install New Software öffnen
    Menü: Help → Install New Software...
  3. Update Site hinzufügen
    Auf Add... klicken und die URL eingeben:
    https://naturzukunft.codeberg.page/jdt-mcp-server/
  4. Plugin auswählen
    "Eclipse JDT MCP Server" anhaken und Next klicken
  5. Installation abschließen
    Lizenz akzeptieren, installieren und Eclipse neu starten
  6. MCP Server aktivieren
    Window → Preferences → JDT MCP Server
    "Enable MCP Server" aktivieren

KI-Assistent konfigurieren

Claude Code

In ~/.claude.json oder Projekt-.claude.json:

{
  "mcpServers": {
    "eclipse-jdt": {
      "url": "http://localhost:51000/sse",
      "transport": "sse"
    }
  }
}

Cursor

In .cursor/mcp.json:

{
  "mcpServers": {
    "eclipse-jdt": {
      "url": "http://localhost:51000/sse",
      "transport": "sse"
    }
  }
}

Hinweis: Der tatsächliche Port wird in der Eclipse-Konsole angezeigt (Standard: 51000-51100).
